Java and Swift App Development at Northshore High School

About Panther Programs

Panther Programs is a new programming club at Northshore High School in Slidell, Louisiana. Starting in August 2016, the club will prepare anyone interested in learning how to make applications for iOS and/or Android. It will start by teaching the fundamentals of two programming languages, Java and Swift.
